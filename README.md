# 💰 Finance Manager — Система управления личными финансами

**Backend-приложение** для управления личными финансами с полным функционалом учета доходов и расходов, бюджетирования, аналитики и переводов между пользователями.

---

## 🚀 Возможности

### 📊 Основной функционал
- ✅ Регистрация и авторизация пользователей
- ✅ Учет доходов и расходов по категориям
- ✅ Бюджетирование — установка лимитов на категории расходов
- ✅ Переводы между пользователями
- ✅ Автосохранение данных при выходе

### 📈 Аналитика и отчетность
- ✅ Детальная статистика по доходам и расходам
- ✅ Фильтрация данных по периодам и категориям
- ✅ Экспорт данных в CSV и JSON форматах
- ✅ Расширенные оповещения о превышении бюджета

### 🔔 Умные уведомления
- ✅ Превышение бюджета — уведомления при превышении лимитов
- ✅ 80% лимита — предупреждение при приближении к лимиту
- ✅ Отрицательный баланс — предупреждение о перерасходе
- ✅ Нулевой баланс — информация о нулевом остатке

## 📦 Установка и запуск

### Требования
- Java 17 или выше
- Maven 3.6+

### Сборка и запуск
```bash
# Клонирование репозитория
git clone https://github.com/yourusername/finance-manager.git
cd finance-manager

# Сборка проекта
mvn clean compile

# Запуск приложения
mvn exec:java -Dexec.mainClass="com.finance.Main"

# Или запуск из собранного JAR
java -cp target/classes com.finance.Main

Тестирование

# Запуск unit-тестов
mvn test

# Запуск ручных тестов
java -cp target/classes com.finance.ManualTest
java -cp target/classes com.finance.FinanceTest

# Проверка стиля кода
mvn checkstyle:check

📋 Команды приложения
Меню авторизации
🔐 Вход — вход в систему по логину и паролю
👤 Регистрация — создание нового пользователя
🚪 Выход — завершение работы приложения

Основное меню
💰 Добавить доход — запись дохода с указанием категории, суммы и описания
💸 Добавить расход — запись расхода с указанием категории, суммы и описания
📊 Установить бюджет — установка/изменение лимита для категории расходов
📈 Показать статистику — полный финансовый отчет
🔍 Фильтр и анализ — аналитика по периодам и категориям
📤 Экспорт данных — выгрузка данных в CSV и JSON
🔄 Переводы — перевод денег между пользователями
❓ Помощь — справочная информация
🚪 Выход — выход из аккаунта с сохранением данных

2024-01-16 18:45;Еда;Расход;1500.00;Продукты

JSON формат (резервное копирование)

{

  "balance": 48500.00,
  "transactions": [

  ]
